基础(二)语法基础

2020-03-21T05:00:00.000Z2020-03-23T07:24:28.000Z5.1k20 分钟中国

1. 条件语句

1.1 if 语句

if 语句用于基于不同的条件来执行不同的动作。

1.2 switch 语句

switch 语句用于基于不同的条件来执行不同的动作,判断变量是否等于某个常量的值，即case只能接常量。

2. 循环语句

2.1 while 循环

while 循环在给定条件为 true 时重复语句或语句组。它会在执行循环主体之前测试条件。

2.2 do…while 循环

do…while 循环与 while 循环类似，但是 do…while 循环会在执行循环主体之后测试条件。

2.3 for 循环

for 循环用于多次执行一个语句序列，简化管理循环变量的代码。
初始化表达式、条件表达式、增量表达式都可以为空，即为死循环。

  • 初始化表达式：在循环开始前执行，通常用于初始化循环变量。
  • 条件表达式：在每次循环开始前执行，用于判断是否继续执行循环体。
  • 增量表达式：在每次循环结束后执行(即i++和++i对循环无影响)，用于更新循环变量。

2.4 循环控制语句

循环控制语句改变代码的执行顺序。通过它可以实现代码的跳转。

控制语句 描述
break 终止循环或 switch 语句，程序流将继续执行紧接着循环或 switch 的下一条语句。
continue 告诉一个循环体立刻停止本次循环迭代，重新开始下次循环迭代。
goto 将控制转移到被标记的语句。但是不建议在程序中使用 goto 语句。

3. C# 数组（Array）

数组是一个存储相同类型元素的固定大小的顺序集合，数组中某个指定的元素是通过索引来访问的。
所有的数组都是由连续的内存位置组成的，数组名指向这段连续空间的首地址。最低的地址对应第一个元素，最高的地址对应最后一个元素。

3.1 一维数组

3.1.1 基本语法

声明数组
在 C# 中声明一个数组，可以使用
datatype[] arrayName;即 数据类型[] 数组名称;

初始化数组
仅声明一个数组时不会在内存中初始化数组,数组是一个引用类型，所以需要使用 new 关键字来创建数组的实例进行初始化数组变量。
arr = new int[10];即 数组名称 = new 数据类型[数组长度];
声明、初始化二合一：
int[] arr = new int[10];即 数据类型[] 数组名称 = new 数据类型[数组长度];

数组赋值
通过使用索引号赋值给一个单独的数组元素：
arr[0] = 100;即 数组名称[索引/下标] = 值;
也可以在声明、初始化数组的同时给数组赋值：
int[] arr = {1, 2, 3, 4, 5};即 数据类型[] 数组名称 = {值1, 值2, 值3, ...};
或者
int[] arr = new int[]{1, 2, 3, 4, 5};即 数据类型[] 数组名称 = new 数据类型[]{值1, 值2, 值3,...};

初始值
数据类型[] 数组名称 = new 数据类型[数组长度]初始化一个数组时，C# 编译器会根据数组类型隐式初始化每个数组元素为一个默认值。
int 数组的所有元素默认值为： 0
float 数组的所有元素默认值为： 0
double 数组的所有元素默认值为： 0.0
string 数组的所有元素默认值为： null
bool 数组的所有元素默认值为： false

访问数组元素
数组元素是通过索引访问的。数组索引从 0 开始，所以索引值从 0 到 arrayName.Length-1。

3.1.2 元素遍历

for 循环遍历数组元素

foreach 循环遍历数组元素

3.1.3 数组元素增、删、改、查

数组元素的增加和删除需要使用数组的复制方法。
数组元素的修改和查询可以直接使用数组的索引。

3.2 多维数组

C# 支持多维数组。多维数组最简单的形式是二维数组。
二维数组是以矩形布局的表中的值。
声明二维数组：
datatype[,] arrayName;即 数据类型[,] 数组名称;
初始化二维数组：
arrayName = new datatype[m,n];即 数组名称 = new 数据类型[数组长度1,数组长度2];
声明、初始化二合一：
datatype[,] arrayName = new datatype[m,n];即 数据类型[,] 数组名称 = new 数据类型[数组长度1,数组长度2];

3.3 交错数组

交错数组是数组的数组。
交错数组是数组的数组。交错数组是一个数组，它的元素是数组。
声明交错数组：
datatype[][] arrayName;即 数据类型[][] 数组名称;
初始化交错数组：
arrayName = new datatype[size][];即 数组名称 = new 数据类型[数组长度1][];
声明、初始化二合一：
datatype[][] arrayName = new datatype[size][];即 数据类型[][] 数组名称 = new 数据类型[数组长度1][];

4. C#函数

C# 函数是一组执行特定任务的语句(位于class或struct语句块中)。函数可以接受参数，也可以返回值。C# 函数的声明和调用都非常简单。

4.1 函数声明

C# 函数声明语法格式如下：

访问修饰符：
访问修饰符定义了一个类、方法、属性或其他成员的范围和可见性。可以使用的访问修饰符如下所示：

  • public：公共访问修饰符。公共访问修饰符允许对该类、方法、属性或其他成员从任何其他类访问。
  • private：私有访问修饰符。私有访问修饰符允许对该类、方法、属性或其他成员从同一类内部访问。
  • protected：受保护的访问修饰符。受保护的访问修饰符允许对该类、方法、属性或其他成员从同一类内部和派生类内部访问。
  • internal：内部访问修饰符。内部访问修饰符允许对该类、方法、属性或其他成员从同一程序集中的任何类访问。
  • protected internal：受保护的内部访问修饰符。受保护的内部访问修饰符允许对该类、方法、属性或其他成员从同一程序集中的任何类和派生类内部访问。
    返回类型：
    返回类型指定函数返回的值的数据类型。如果函数不返回任何值，则返回类型为 void。
    函数名：
    函数名是一个标识符，用于标识函数。函数名必须遵循Pascal命名规则，即首字母大写，后面的单词首字母大写。
    参数列表：
    参数列表指定函数接受的参数的类型和名称(形参)。参数列表用圆括号括起来，参数之间用逗号分隔。如果函数不接受任何参数，则参数列表为空；如果需要参数，以参数类型 参数名 的形形式书写。
    函数体：
    函数体包含了函数执行的代码。函数体用花括号括起来。

4.3高级参数

函数参数传递有值传递和引用传递两种方式：

值传递

值传递是指将参数的值复制给函数的形式参数。在这种情况下，函数的形式参数和实际参数是不同的变量，它们的值是独立的。
调用函数时，会将函数加载到栈空间，将实际参数的值复制给形式参数，函数内部对形式参数的修改不会影响实际参数的值。

引用传递(ref和out)

  • ref 关键字用于传入值类型参数时，需要让在函数内部进行的值操作影响到实际参数的值。
  • out 关键字用于传入引用类型参数时，需要让在在函数内进行重新声明时也能影响到实际参数的值

引用传递是指将参数的引用（地址）传递给函数的形式参数。在这种情况下，函数的形式参数和实际参数是同一个变量，它们的值是相同的。
C# 中默认的参数传递方式是值传递。如果需要引用传递，可以使用 ref 关键字。
ref 关键字用于声明引用参数。引用参数必须在调用函数之前进行初始化。
调用函数时，会将函数加载到栈空间，将实际参数的地址复制给形式参数，函数内部对形式参数的修改会影响实际参数的值。

4.4递归调用

递归调用是指函数在执行过程中调用自身的过程。递归调用可以用于解决一些复杂的问题，例如计算阶乘、斐波那契数列等。
正确的递归调用需要满足以下条件：

  • 递归终止条件：递归调用必须有一个终止条件，即当满足该条件时，递归调用将停止。
  • 递归调用的变化：递归调用必须在每次调用时改变进行终止条件判断的参数值，以便在下次调用时达到终止条件。

5. 基础排序算法

5.1 冒泡排序

冒泡排序是一种简单的排序算法，它重复地走访过要排序的数列，一次比较两个元素，如果它们的顺序错误就把它们交换过来。
冒泡排序的基本思想是：

  • 比较相邻的元素。如果第一个比第二个大，就交换它们两个；
  • 对每一对相邻元素作同样的工作，从开始第一对到结尾的最后一对，这样在最后的元素应该会是最大的数；
  • 针对所有的元素重复以上的步骤，除了最后一个；
  • 重复步骤1~3，直到排序完成。
    冒泡排序的时间复杂度为 O(n^2)，空间复杂度为 O(1)。

5.2 选择排序

选择排序是一种简单直观的排序算法，它的工作原理是：

  • 首先在未排序序列中找到最小（大）元素，存放到排序序列的起始位置；
  • 再从剩余未排序元素中继续寻找最小（大）元素，然后放到已排序序列的末尾；
  • 重复第二步，直到所有元素均排序完毕。
    选择排序的时间复杂度为 O(n^2)，空间复杂度为 O(1)。
